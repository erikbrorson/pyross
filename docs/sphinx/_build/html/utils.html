Utils¶

Miscellaneous classes and methods used in PyRoss which do not fit elsewhere.

utils.getPopulation()¶

Takes coutry name and number, M, of age-groups as argument Returns population structured in M age-groups

Parameters

  • country (string) – Default is ‘India’

  • M (int) – Deafault is 16 age-groups

utils.hessian_finite_difference()¶

Forward finite-difference computation of the Hessian of a function.

Parameters

  • pos (float or numpy.array(dims=1), optional) – Position at which the hessian is to be computed.

  • function (function(numpy.array)) – Function of interest.

  • pos – Step size used for FD computation (can be parameter dependant).

  • method (str) – Different options for the FD computation: “forward” or “central”.

Returns

hess – Hessian of function at pos.

Return type

numpy.array(dims=2)
